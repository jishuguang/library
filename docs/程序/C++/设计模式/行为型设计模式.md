## Strategy

### 适用场景

算法流程一致，具体实现存在不同策略。

```{note}
与模板方法比较。
```

### 运行时策略模式

```cpp
class Strategy {
    virtual void A();
    virtual void B();
};

class StrategyA : public Strategy;
class StrategyB : public Strategy;

class Client {
public:
    Client(shared_ptr<Strategy> strategy);
    void Func()
    {
        strategy->A();
        strategy->B();
    }
private:
    shared_ptr<Strategy> strategy;
};
```

### 编译时策略模式

```cpp
class Strategy;

template<typename T>
class Client {
public:
    void Func()
    {
        // strategy method call
    }    
private:
    T strategy;    
};
```

## State

```{note}
补充分析Boost状态机实现。
```

### 适用场景

状态机。

### 典型实现

```cpp
class State;
class StateHolder {
public:
    StateHolder(shared_ptr<State> state);
    SetState(shared_ptr<State> state);
private:
    shared_ptr<State> state;
};

class State {
public:
    virtual void ActionA(StateHolder& holder) {}
    virtual void ActionB(StateHolder& holder) {}
};

class State1 : public State {
public:
    void ActionA(StateHolder& holder) override
    { 
        // do something
        // change state or keep state
    }

    // ActionB may not be overrided
};
```

### 枚举实现

```cpp
enum class State {
    ONE,
    TWO,
    MAX
};

enum class Action {
    A,
    B,
    MAX
};

class StateHolder {
public:
    void TransState(Action& action) 
    {
        // code to check rules
        state = rules[state][action];
    }    
private:
    static const map<State, map<Action, State>> rules = {
        {State::ONE, 
            { 
                {Action::A, State::TWO},
                {Action::B, State::ONE}
            }
        }
    }
    State state = State::ONE;
};
```

## Obeserver

### 适用场景

订阅、通知。

### 典型实现

```cpp
class Msg;

template<typename T>
class Observer {
    virtual void Handle(T& source, Msg& msg) {}
};

template<typename T>
class Observable {
public:
    void Subscribe(shared_ptr<Observer<T> obs)
    {
        lock_guard<mutex> lock<obsLock>;
        // add
    }

    void Unubscribe(shared_ptr<Observer<T> obs)
    {
        lock_guard<mutex> lock<obsLock>;
        // remove
    }

    void Notify(T& source, Msg& msg)
    {
        lock_guard<mutex> lock<obsLock>;
        for (auto& obs: observers) {
            obs->Handle(T, msg);
        }
    }
private:
    set<shared_ptr<Observer<T>>> observers;  // 避免重复订阅
    mutex obsLock;
}

class A : public Observable<A>;
class B : public Observer<A>;
```

### 注意点

```{note}
注意Observable的可重入性。
```

## Chain of Responsibility

### 适用场景

顺序处理。

### 典型实现

```cpp
class Request;

class Filter {
public:
public:
    void SetNext(shared_ptr<Filter> filter)
    {
        if (next) {
            next->SetNext(filter);
        } else {
            next = filter;
        }
    }

    virtual void Handle(Request& request) 
    {
        // Handle request

        if (next) {
            next->Handle(request);
        }
    }

private:
    shared_ptr<Filter> next;
};

class AFilter : public Filter {};
```

## Commnad

### 适用场景

封装命令，提供额外功能，例如：审计、回退等。

### 典型实现

```cpp
class Command {
    virtual void do() {};
    virtual void undo() {};
};

class CompositeCommand : public Command {
public:
    void AddCommand(shared_ptr<Command> command);
    void do() 
    {
        for (auto& cmd: commands) {
            cmd->do();
        }
    }

    void undo()
    {
        for (auto it = commands.rbegin(); it != commands.rend(); ++it) {
            (*it)->undo();
        }
    }
private:
    vector<shared_ptr<Command>> commands;    
}
```
