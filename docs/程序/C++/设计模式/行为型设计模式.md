## Obeserver

### 适用场景

订阅、通知。

### 典型实现

```cpp
class Msg;

template<typename T>
class Observer {
    virtual void Handle(T& source, Msg& msg) {}
};

template<typename T>
class Observable {
public:
    void Subscribe(shared_ptr<Observer<T> obs)
    {
        lock_guard<mutex> lock<obsLock>;
        // add
    }

    void Unubscribe(shared_ptr<Observer<T> obs)
    {
        lock_guard<mutex> lock<obsLock>;
        // remove
    }

    void Notify(T& source, Msg& msg)
    {
        lock_guard<mutex> lock<obsLock>;
        for (auto& obs: observers) {
            obs->Handle(T, msg);
        }
    }
private:
    set<shared_ptr<Observer<T>>> observers;  // 避免重复订阅
    mutex obsLock;
}

class A : public Observable<A>;
class B : public Observer<A>;
```

### 注意点

```{note}
注意Observable的可重入性。
```

## Chain of Responsibility

### 适用场景

顺序处理。

### 典型实现

```cpp
class Request;

class Filter {
public:
public:
    void SetNext(shared_ptr<Filter> filter)
    {
        if (next) {
            next->SetNext(filter);
        } else {
            next = filter;
        }
    }

    virtual void Handle(Request& request) 
    {
        // Handle request

        if (next) {
            next->Handle(request);
        }
    }

private:
    shared_ptr<Filter> next;
};

class AFilter : public Filter {};
```

## Commnad

### 适用场景

封装命令，提供额外功能，例如：审计、回退等。

### 典型实现

```cpp
class Command {
    virtual void do() {};
    virtual void undo() {};
};

class CompositeCommand : public Command {
public:
    void AddCommand(shared_ptr<Command> command);
    void do() 
    {
        for (auto& cmd: commands) {
            cmd->do();
        }
    }

    void undo()
    {
        for (auto it = commands.rbegin(); it != commands.rend(); ++it) {
            (*it)->undo();
        }
    }
private:
    vector<shared_ptr<Command>> commands;    
}
```
