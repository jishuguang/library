# C++习惯用法

## CRTP

Curiously Recurring Template Pattern

```cpp
template<class Z>
class Y {};
 
class X : public Y<X> {};
```

可用于实现编译时多态、装饰器模式等。示例：

- 标准库[enable_shared_from_this](https://en.cppreference.com/w/cpp/memory/enable_shared_from_this)
